## BACKEND 

I built it using **Node.js**, **Express**, and **MongoDB (Atlas)**, and deployed on **Render**.

### Lessons
- Fetchs all lessons  
- Searchs lessons (name, subject, location)
- Updates all lesson space count  

### Orders
- Saves new orders into MongoDB  
- Each booking automatically reduces lesson space  

### CORS Enabled
Allows frontend hosted on GitHub Pages to access the backend
**:)**

## Technologies Used
- **Node.js**
- **Express.js**
- **MongoDB (Atlas)**
- **Render.com for deployment**
- **logger middleware**
